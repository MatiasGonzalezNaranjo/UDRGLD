Operadores de asignación: ayuda a asigbar un valor a una variable, se utiliza "=" para asignar un valor, "+=" para sumar cierto valor a una variable, "*=" para multiplicar la variable, "/=" para realizar una division, "%=" se realiza la divicion, resta y se coloca el residuo, "**=" asi se representa la potenciacion
operadores de comparación: "==" compara si dos variables son iguales, "===" compara de forma estricta si dos variables son iguales, teniendo en cuenta el tipo de dato, "!" cuestiona si los valores son distintos o no, ">" cuestiona si es mayor que, "<" cuestiona si es menor que, ">=" cuestiona si es mayor o igual, "<=" cuestiona si es menor o igual
Operadores logicos: "&&" significa "Y" necesita tener las dos variables verdaderas para que salga true, "||" significa "Ó" y necesita tener almenos una de las dos variables verdaderas para que salga true, "!" se utilliza como negación, cambia el valor de el dato buleano al contrario
operador ternario: se divide en 3 segmentos, primeramente se tiene que realizar las operaciones y definiciones de las variables, cumplido el primer paso se realiza la operacion que se nos propone con un signo de interrogacion "?", este segundo paso nos puede dar como resultado un dato buleano, en caso de ser verdadero, la respuesta es la que va a la isquierda de los puntos ":" y en caso de ser falso el lado derecho
